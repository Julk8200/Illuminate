## 🛠️ How to Download and Install Illuminate on macOS

Follow these steps to get Illuminate up and running:

1. **Open the Download Page**  
   Visit https://github.com/Julk8200/Illuminate in your web browser.

2. **Go to the Releases Section**  
   Scroll down or look to the right sidebar for a link called **Releases**. Click it.

3. **Find the Latest Release**  
   The latest release includes the newest files for downloading. Look for the most recent version number, usually at the top.

4. **Download the macOS App File**  
   Find a file with a name similar to `Illuminate-macos.dmg` or `Illuminate.app.zip`. Click the file name to start downloading it to your computer.

5. **Open the Downloaded File**  
   Once the download finishes, locate the file in your **Downloads** folder.

6. **Install the App**  
   - If it is a `.dmg` file: Double-click to open it. A window will appear showing the app icon. Drag the **Illuminate** icon into your **Applications** folder.  
   - If it is a `.zip` file: Double-click it to unpack. Move the extracted **Illuminate.app** to the **Applications** folder.

7. **Launch Illuminate**  
   Open your **Applications** folder, find **Illuminate**, and double-click to open the browser.

8. **Grant Permissions if Needed**  
   macOS may ask if you trust the app since it comes from the internet. Click **Open** to approve running the app.
